# 隐私权政策

## 引言

感谢您使用我们的谷歌插件。本隐私权政策解释了我们如何收集、使用和保护您的信息。请仔细阅读本政策，以了解我们的做法。

## 我们收集的信息

### 用户活动

我们可能会收集您在使用插件时的活动信息，包括但不限于：

- 您访问的网页及其内容
- 您与网页进行交互的方式

### 网页内容

我们插件可能会收集您正在访问网页的内容信息，以便提供相关功能和服务。具体包括：

- 网页标题和 URL
- 网页内容的特定部分（如文本、图片）

## 信息的使用方式

我们收集的信息将用于以下目的：

- 提供和改进插件的功能
- 个性化用户体验
- 分析和研究用户行为，以改善我们的服务

## 信息的共享

我们不会与第三方共享您的个人信息，除非在以下情况下：

- 获得您的明确同意
- 法律要求我们这样做
- 为了保护我们的权利和财产

## 信息的保护

我们采取合理的技术和组织措施，保护您的信息免受未经授权的访问、使用或披露。这些措施包括但不限于：

- 数据加密
- 定期安全审计
